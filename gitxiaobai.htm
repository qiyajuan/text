一、         Git安装

1、     下载国内镜像文件 https://pan.baidu.com/s/1kU5OCOB#list/path=%2Fpub%2Fgit

2、     安装完成后，在开始菜单里找到“Git”->“Git Bash”，蹦出一个类似命令行窗口的东西，就说明Git安装成功！

3、     安装完成后，还需要最后一步设置，在命令行输入：

$ git config --global user.name "Your Name"

$ git config --global user.email "email@example.com"

二、         登录Github创建仓库

1、打开网址https://github.com/ 登录

2、页面左上角，点击“New”，如图：

4、     开始创建仓库，具体操作如图：

5、     点击创建后，进入仓库界面，点击右侧绿色按钮，如图复制链接备用

  

三、         上传本地文件到Github

1、     在本地任意一个盘，例如我在D盘，右击选择Git Bash Here，如图：

 

2、     输入指令：git clone https://github.com/qiyajuan/text.git  //clone后面的内容即为之前的备用连接

目的是将Github仓库里的text导入本地文件夹，如图：

3、     这样本地D盘里就多一个text文件夹

  

4、     接下来举例，新建一个Word文档，自己随便编辑，另存为html文件，编码改成UTF-8

5、     将生成的.html和.files两个文件复制到你clone的text文件夹里面

    

6、     再次打开命令框，输入指令（如果命令框关闭，则再次重复本标题下1步骤即可继续）

cd text     (注：此操作进入text文件夹内)

git add .   （注：别忘记后面的. 此操作是把Test文件夹下面的文件都添加进来）

git commit  -m  "提交信息"  （注：“提交信息”里面换成你需要，如“first commit”）

git push -u origin master   （注：此操作目的是把本地仓库push到github上面，此步骤需要你输入帐号和密码）

7、     出现下图就好了

8、     回到Github的界面刷新一下，复制进本地文件夹的文件就会出现在仓库里

四、         访问仓库里的文件

1、     点击文件名下方的Setting，如图：

2、     向下拉找到GitHub Pages，将Source下的None换选成master branch

3、     界面会自动刷新，再下拉到Github Pages，会出现一个网址

4、     这个网址只是text文件夹的地址，如果我们需要打开test.html这个文件，则需要在浏览器中补全输入

https://qiyajuan.github.io/text/test.html，这样文件就会以网页的形式出现！
